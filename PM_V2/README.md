## DevContainer 環境

PM作業（WBS作成、スケジュール管理、要件定義、設計ドキュメント作成）に特化したDevContainer環境を提供しています。

### 含まれるツール

| ツール | 用途 |
|--------|------|
| Mermaid (CLI + VSCode拡張) | ガントチャート、WBS図、フローチャート、ER図、シーケンス図 |
| Draw.io (VSCode拡張) | 自由なダイアグラム作成 |
| PlantUML | UML図の作成 |
| Python (pandas, openpyxl, matplotlib) | Excel操作、EVM分析、コスト計算 |
| Markdown Preview Enhanced | Markdown→PDF変換（日本語対応） |

### 使い方

1. VSCodeで `PM_V2/` フォルダを開く
2. コマンドパレットから「Dev Containers: Open Folder in Container」を選択
3. `PM_V2/` を指定してコンテナを起動

### EVM分析ツール

```bash
# CSV/Excelからの EVM 指標計算
python scripts/evm_calculator.py data.csv

# S曲線チャート出力
python scripts/evm_calculator.py data.csv --chart evm_chart.png
```

入力ファイルには `period`, `PV`, `EV`, `AC` カラムが必要です。
