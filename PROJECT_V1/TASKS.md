# Project V1

## 開発工程ごとのタスク整理

- [ ] 開発工程内のタスクを整理 → docs フォルダ内のdev_process.md に記載する （IPAのドキュメントに基づく）
- [ ] dev_process.md に開発工程ごとに必要なツールまたはIPAの情報を参照する
- [ ] docs/IPA内のPDFを参照してすべての開発工程のタスクを精密化して。
- [ ] その他、IPAのURLを参照してどのようなタスクがあるかを見つけて。
- [ ] dev_process.md をベースに別のMarkdownファイルに開発工程ごとに必要なツールを100件以上。最も工程内のタスクごとに一般的で生産性が高いものを5つ以上選定。
