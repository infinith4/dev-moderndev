# プロジェクト要件チェックリスト

**作成日**: 2025-12-15
**バージョン**: 1.0
**目的**: プロジェクトで必要な作業項目・技術スタックを選択するための質問形式チェックリスト

---

## 使い方

このチェックリストは、プロジェクトの開発工程ごとに必要な作業項目と技術スタックを選択するためのものです。

1. 各工程で必要な作業項目にチェック `[x]` を入れてください
2. 技術スタック選択では、使用する技術に `[x]` を入れるか、選択肢の後に具体的な内容を記入してください
3. プロジェクトに該当しない項目は空欄 `[ ]` のままで構いません

## チェックリスト完了後の次のステップ

このチェックリストを完了したら、以下のステップを実施してください:

1. **選択項目の集計**: チェックした項目を工程別・カテゴリ別に集計
2. **技術スタック選定資料の作成**: 選択した技術の詳細調査・比較検討
3. **プロジェクト計画への反映**: 必要な作業項目をWBSやタスクリストに展開
4. **見積もり**: 選択した作業項目に基づいた工数・コスト見積もり
5. **承認プロセス**: ステークホルダーへの説明・承認取得
