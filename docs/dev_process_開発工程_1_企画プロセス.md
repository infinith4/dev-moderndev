## 1. 概要

企画プロセスのタスクと推奨ツール、有用なドキュメントを記載した。

---

### 1.2. 共通

**対応項目**
- 経営課題・業務課題の整理
- システム化対象範囲の定義
- 投資対効果・実現性の評価
- 企画承認に向けた計画と合意形成

**有用なドキュメント**

| 資料名 | 用途 |
|-------|------|
| [DX推進指標](https://www.ipa.go.jp/digital/dx/dx-suishin/about.html) | 現状分析、課題整理、企画方針立案 |
| [ソフトウェア開発見積りガイドブック](https://www.ipa.go.jp/archive/publish/secbooks20060425.html) | 概算見積り、工数・期間の初期試算 |
| [共通フレーム2013](https://www.ipa.go.jp/archive/files/000027415.pdf) | 企画〜運用までの工程定義、成果物整理 |

## 2. システム化構想の立案
**成果物**
- システム化構想書
- 課題整理資料
- システム化対象業務一覧
- 構想レベル業務フロー

| ツール名 | 用途 |
|---------|------|
| [Notion](https://www.notion.so/) | 構想メモ、検討ログ、レビュー記録の集約 |
| [Draw.io](https://www.diagrams.net/) | 現行/将来業務フロー、構想図の可視化 |
| [Miro](https://miro.com/) | 課題整理、アイデア整理、関係者合意形成 |

**有用なドキュメント**

| 資料名 | 用途 |
|-------|------|
| [デジタル・ガバメント推進標準ガイドライン](https://www.digital.go.jp/resources/standard_guidelines/) | 構想段階の全体方針、業務見直し観点の整理 |
| [BABOK Guide (概要)](https://www.iiba.org/career-resources/a-business-analysis-professionals-foundation-for-success/babok/) | 要求整理、課題分析、利害関係者分析 |

## 3. システム化計画の立案
**成果物**
- システム化計画書
- 概算費用試算
- スケジュール計画
- リスク一覧

| ツール名 | 用途 |
|---------|------|
| [Microsoft Excel](https://www.microsoft.com/microsoft-365/excel) | 概算見積り、WBS、費用試算 |
| [Redmine](https://www.redmine.org/) | 課題管理、進捗管理、体制管理 |
| [Draw.io](https://www.diagrams.net/) | システム全体構成、ロードマップ可視化 |

**有用なドキュメント**

| 資料名 | 用途 |
|-------|------|
| [ソフトウェア開発見積りガイドブック](https://www.ipa.go.jp/archive/publish/secbooks20060425.html) | 計画フェーズの見積り、要員計画 |
| [アジャイル開発版モデル取引・契約書](https://www.ipa.go.jp/digital/model/agile20200331.html) | 契約方針、役割分担、計画時の合意事項整理 |
| [PMBOK Guide (概要)](https://www.pmi.org/pmbok-guide-standards/foundational/pmbok) | スケジュール、コスト、リスク管理計画 |
