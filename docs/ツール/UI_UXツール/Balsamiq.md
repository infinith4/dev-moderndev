# Balsamiq

## 概要

Balsamiqは、手描き風のUIワイヤーフレーム作成に特化したプロトタイピングツールです。「ラピッドローファイワイヤーフレーミング」をコンセプトに、細部にこだわらず素早くアイデアをスケッチできます。シンプルなインターフェースにより、デザイナーだけでなく非デザイナーも直感的にUI設計でき、ステークホルダーとのコミュニケーションツールとして広く利用されています。

## 主な機能

### 1. 手描き風スケッチ
- **ローファイ**: 手描き風のラフなデザイン
- **フォーカス**: 構造・機能に集中
- **非完成感**: 議論を促進
- **ラピッド**: 素早いプロトタイピング

### 2. 豊富なUIコンポーネント
- **フォーム**: ボタン、テキストボックス、ドロップダウン
- **ナビゲーション**: メニュー、タブ、ブレッドクラム
- **メディア**: 画像プレースホルダー、動画、アイコン
- **レイアウト**: グリッド、パネル、カード
- **モバイル**: iOS、Androidコンポーネント

### 3. リンク・インタラクション
- **リンク設定**: 画面遷移
- **プレゼンテーションモード**: クリッカブルプロトタイプ
- **ノート**: デザインメモ

### 4. コラボレーション
- **コメント**: フィードバック機能
- **バージョン履歴**: 変更追跡
- **共有**: リンク共有、PNG/PDFエクスポート

### 5. テンプレート・アセット
- **テンプレート**: 一般的なUI パターン
- **シンボル**: 再利用可能なコンポーネント
- **アイコン**: Font Awesome統合

## 利用方法

### インストール

```
# デスクトップアプリ
https://balsamiq.com/wireframes/desktop/ からダウンロード

# クラウド版
https://balsamiq.cloud にアクセス

# プラグイン
- Confluence用
- Jira用
- Google Drive用
```

### 基本操作

```
1. 新規プロジェクト作成
   File → New Project

2. ワイヤーフレーム作成
   + New Wireframe

3. UIコンポーネント追加
   左パネル → UI Library
   - Button: ドラッグ&ドロップ
   - Text Input: ドラッグ&ドロップ
   - Label: テキスト追加

4. リンク設定
   コンポーネント選択 → Link To → 遷移先画面選択

5. プレゼンテーションモード
   上部メニュー → Presentation Mode
   クリックして画面遷移確認
```

### ログイン画面例

```
コンポーネント配置:
1. Label: "Login to Your Account"
2. Text Input: "Email"
3. Password Input: "Password"
4. Checkbox: "Remember me"
5. Button: "Sign In"
6. Link: "Forgot password?"
7. Button: "Create Account"
```

### ダッシュボード例

```
コンポーネント配置:
1. Top Navigation: メニューバー
2. Sidebar: サイドバーナビゲーション
3. Breadcrumbs: パンくずリスト
4. Data Table: ユーザー一覧
5. Charts: 売上グラフ
6. Button Group: アクション
```

### エクスポート

```
# PNG画像
File → Export → Export to PNG
- 全画面または選択画面

# PDF
File → Export → Export to PDF
- プロジェクト全体またはワイヤーフレーム

# BMML (Balsamiq形式)
File → Export → Export to BMML XML
```

### Confluence統合

```
1. Confluenceページ編集
2. "/" → Balsamiq Wireframes
3. マクロ挿入
4. ワイヤーフレーム作成
5. ページ保存
```

## メリット

### ✅ 主な利点

1. **ラピッド**: 素早いワイヤーフレーム作成
2. **手描き風**: ローファイ・非完成感
3. **直感的**: 学習曲線が緩やか
4. **非デザイナー向け**: 誰でも使える
5. **豊富なコンポーネント**: UIライブラリ充実
6. **クロスプラットフォーム**: Windows、Mac、Web
7. **統合**: Confluence、Jira、Google Drive
8. **軽量**: 高速動作
9. **オフライン**: デスクトップ版で利用可能
10. **買い切り**: デスクトップ版は永続ライセンス

## デメリット

### ❌ 制約・課題

1. **ローファイのみ**: ハイファイデザイン不可
2. **インタラクション制限**: 複雑なプロトタイプ不可
3. **デザイン変更**: 手描き風のみ
4. **チーム協業**: リアルタイム同時編集不可
5. **バージョン管理**: Git統合なし
6. **開発者ハンドオフ**: Figmaほどの機能なし
7. **モバイルアプリ**: アプリ版なし
8. **アニメーション**: 動的プロトタイプ不可

| **Adobe XD** | Adobe統合、ハイファイ | Balsamiqよりデザイン詳細 |
