# Figma

## 概要

Figmaは、ブラウザベースのUI/UXデザインツールです。Webアプリケーション、モバイルアプリ、デスクトップアプリのインターフェースデザイン、プロトタイプ作成、デザインシステム構築を一つのプラットフォームで実現します。クラウドネイティブな設計により、リアルタイム協業、バージョン管理、デベロッパーハンドオフ機能を備え、世界中で400万人以上のデザイナーと開発者に利用されています。

## 主な機能

### 1. ベクターデザインツール
- ペンツール、シェイプツール
- ベクター編集
- テキストツール（Google Fonts統合）
- 画像配置・マスク
- エフェクト（影、ぼかし、ブレンド）

### 2. コンポーネント・バリアント
- 再利用可能なコンポーネント
- バリアント（状態管理）
- Auto Layout（レスポンシブ対応）
- コンポーネントプロパティ

### 3. プロトタイプ
- 画面遷移の設定
- インタラクション（クリック、ホバー、スクロール）
- アニメーション（イージング、遅延）
- プレビュー・共有

### 4. デザインシステム
- スタイルガイド（色、タイポグラフィ、エフェクト）
- コンポーネントライブラリ
- チーム共有ライブラリ
- バージョン管理

### 5. リアルタイム協業
- 複数人での同時編集
- カーソル表示（誰が編集中か可視化）
- コメント・フィードバック
- 音声通話（FigJam）

### 6. デベロッパーハンドオフ
- CSS、iOS、Android コード自動生成
- アセットエクスポート（PNG、SVG、PDF）
- スペーシング・サイズ表示
- プラグイン（Zeplin、Avocode代替）

### 7. FigJam（ホワイトボード）
- ブレインストーミング
- ワイヤーフレーム
- ワークショップ
- 付箋、図形、手書き

## 利用方法

### デザイン作成

```
1. 新規ファイル作成
   Dashboard → New Design File

2. フレーム追加
   F キーでFrame Tool
   → iPhone 14 Pro、Desktop HD 等のプリセット選択

3. シェイプ追加
   R: Rectangle（長方形）
   O: Ellipse（楕円）
   T: Text（テキスト）

4. レイヤー編集
   - プロパティパネルで色、サイズ、位置調整
   - エフェクト追加（Drop Shadow等）

5. Auto Layout適用
   要素を選択 → Shift+A
   → レスポンシブ対応

6. コンポーネント化
   再利用する要素を選択 → Cmd/Ctrl+Alt+K
   → マスターコンポーネント作成
```

### プロトタイプ作成

```
1. Prototype タブに切り替え

2. インタラクション設定
   - フレームの端からドラッグ
   - 遷移先フレームに接続
   - トリガー選択（On Click, On Hover等）
   - アニメーション選択（Instant, Dissolve, Slide等）

3. プレビュー
   右上の再生ボタン
   → ブラウザでプロトタイプ確認

4. 共有
   Share → Copy link
   → ステークホルダーに共有
   → コメント収集
```

### デザインシステム構築

```
1. スタイル定義
   - 色: 塗りを選択 → スタイル作成（Primary, Secondary等）
   - タイポグラフィ: テキスト選択 → Text Style作成
   - エフェクト: Effect Style作成

2. コンポーネントライブラリ
   - ボタン、フォーム、カード等をコンポーネント化
   - バリアント作成（Primary/Secondary、Small/Large等）

3. チーム共有
   - ファイルをTeam Libraryとして公開
   - 他プロジェクトで利用可能
   - 更新時は全プロジェクトに反映
```

### デベロッパーハンドオフ

```
1. Inspect パネル（開発者ビュー）
   要素選択 → 右パネルでコード表示

2. CSS/iOS/Android コードコピー
   - Code タブで言語選択
   - コードスニペットをコピー

3. アセットエクスポート
   - 要素選択 → Export設定
   - PNG, SVG, JPG, PDF選択
   - 1x, 2x, 3x（高解像度）対応

4. プラグイン活用
   - Anima: React/HTML/Vue コード生成
   - Figma to Code: 高度なコード生成
```

## メリット

### ✅ 主な利点

1. **ブラウザベース**: インストール不要、どこからでもアクセス
2. **リアルタイム協業**: デザイナー・開発者が同時編集
3. **Auto Layout**: レスポンシブデザインを効率化
4. **コンポーネントシステム**: 再利用性とメンテナンス性向上
5. **無料プラン**: 個人利用なら無料で十分
6. **プロトタイプ**: アニメーション付き遷移
7. **デベロッパーハンドオフ**: コード自動生成
8. **プラグインエコシステム**: 機能拡張が豊富
9. **バージョン管理**: 変更履歴の確認・復元
10. **クロスプラットフォーム**: Windows、Mac、Linux、Web対応

## デメリット

### ❌ 制約・課題

1. **オフライン不可**: インターネット接続必須
2. **無料版制限**: 3ファイルまで（Professionalは無制限）
3. **大規模ファイル重い**: 複雑なデザインでは動作が遅い
4. **Adobeエコシステム**: Illustrator/Photoshop との互換性に課題
5. **ラスター編集弱い**: 写真編集はPhotoshopに劣る
6. **日本語フォント**: 一部フォント非対応
7. **学習曲線**: 高度な機能は習得に時間がかかる
8. **セキュリティ**: クラウド保存のため機密情報管理に注意

## 代替ツール

| ツール | 特徴 | 比較 |
|--------|------|------|
| **Adobe XD** | Adobe製、Adobe CC統合 | Figmaに押され気味、開発停止の噂 |
| **Sketch** | Mac専用、老舗 | Figmaにシェア奪われ中 |
| **Penpot** | オープンソース、無料 | Figmaより機能少ないが無料 |
| **Framer** | プロトタイプ・コード生成重視 | Figmaよりインタラクション強い |
| **InVision Studio** | プロトタイプ特化 | Figmaより機能限定的 |
