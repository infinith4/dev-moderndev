# Pencil Project

## 概要

Pencil Projectは、GUIプロトタイピングとワイヤーフレーム作成のための無料・オープンソースツールです。Windows、macOS、Linuxのクロスプラットフォームで動作し、デスクトップ・モバイル（Android/iOS）の組み込みUIステンシルを備えています。ドラッグ&ドロップでワイヤーフレームを作成し、ページ間リンクでUIフローを定義でき、PDF、SVG、PNG等の形式でエクスポートが可能です。

## 主な機能

### 1. UI設計

- **組み込みステンシル**: デスクトップ（Windows/macOS）、Webフォーム、Android、iOS
- **ドラッグ&ドロップ**: UIコンポーネントをキャンバスに配置
- **カスタムシェイプ**: 独自のステンシルコレクション作成
- **テキスト編集**: リッチテキスト対応のラベル・テキスト配置

### 2. プロトタイピング

- **ページ間リンク**: UIエレメントをクリックして別ページに遷移する定義
- **UIフロー**: 画面遷移フローの可視化
- **複数ページ**: ドキュメント内に複数ページを管理
- **ツリー構造**: ページをツリー形式で階層管理（v3以降）

### 3. エクスポート

- **PDF**: ドキュメント全体のPDF出力
- **SVG**: Inkscape互換のSVGベクターファイル
- **PNG**: ラスター画像出力
- **LibreOffice**: OpenDocument形式でのエクスポート

### 4. リソース管理

- **クリップアート**: キーワード検索でクリップアートをドラッグ&ドロップ
- **外部ビットマップ**: 画像ファイルの取り込み
- **ZIP形式**: 大容量ドキュメントに対応したファイルフォーマット（v3）

## 利用方法

### インストール

```
# ダウンロード
https://pencil.evolus.vn/ から各プラットフォーム用インストーラをダウンロード

# 対応OS
- Windows: インストーラ (.exe)
- macOS: DMGパッケージ
- Linux: DEB / RPM / AppImage
```

### 基本操作

```
1. 新規ドキュメント作成: File > New
2. ステンシル選択: 左パネルからUIコンポーネントを選択
3. 配置: キャンバスにドラッグ&ドロップ
4. プロパティ編集: 右パネルでサイズ、色、テキストを変更
5. ページ追加: Pages パネルで「+」をクリック
6. リンク設定: エレメントのプロパティで「Link to」を指定
7. エクスポート: Document > Export Document
```

### ステンシルコレクション

| コレクション | 内容 |
|-------------|------|
| **General** | 基本図形、矢印、コネクタ |
| **Desktop - Gtk Widgets** | GTKウィジェット |
| **Web UI** | Webフォーム要素（ボタン、入力欄、メニュー） |
| **Android** | Android UIコンポーネント |
| **iOS Wireframe** | iOS UIコンポーネント |
| **Flowchart** | フローチャート図形 |

## メリット

1. **完全無料**: オープンソースで商用利用も可能
2. **クロスプラットフォーム**: Windows、macOS、Linux対応
3. **軽量**: インストールサイズが小さく動作が軽快
4. **組み込みステンシル**: Android/iOSを含むUIコンポーネントが標準搭載
5. **ページリンク**: UIフローの定義が可能
6. **多形式エクスポート**: PDF、SVG、PNG、LibreOffice対応
7. **オフライン動作**: インターネット接続不要

## デメリット

1. **開発停滞**: オープンソースプロジェクトの更新頻度が低い
2. **コラボレーション不可**: リアルタイム共同編集機能がない
3. **高度なプロトタイプ不可**: インタラクティブなアニメーション・遷移効果は非対応
4. **デザインシステム連携なし**: FigmaやSketchのような共有ライブラリ機能がない
5. **ステンシル限定**: 最新のUI/UXトレンドに追従するステンシルが不足
