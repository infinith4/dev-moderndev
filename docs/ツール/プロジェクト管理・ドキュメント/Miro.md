# Miro

## 概要

Miroは、オンラインホワイトボード・ビジュアルコラボレーションプラットフォームです。無限のキャンバス上でチームがリアルタイムにブレインストーミング、プロジェクト計画、ワークショップ、アジャイル開発、デザイン思考を実施できます。豊富なテンプレート、付箋、図形、コネクター、投票機能により、リモート・ハイブリッドチームの協働を促進します。

## 主な機能

### 1. 無限キャンバス
- **無制限の作業スペース**: 無限に広がるホワイトボード
- **ズーム・パン**: シームレスなナビゲーション
- **フレーム機能**: 領域を区切って整理
- **グリッド・ガイド**: 整列・配置補助

### 2. リアルタイム協業
- **同時編集**: 複数ユーザーが同時に作業
- **カーソル表示**: 他のメンバーの操作を可視化
- **@メンション**: チームメンバーへの通知
- **コメント**: オブジェクトへのコメント追加
- **ビデオチャット**: Zoom、Teams、Google Meet統合

### 3. 豊富なオブジェクト
- **付箋**: ブレインストーミング
- **図形**: フローチャート、マインドマップ
- **テキスト**: 見出し、本文
- **画像・ファイル**: ドラッグ&ドロップ
- **リンク**: Webページ、Jira、Confluence連携
- **コネクター**: オブジェクト間の関連

### 4. テンプレート
- **ブレインストーミング**: アイデア整理
- **カンバンボード**: タスク管理
- **ユーザーストーリーマップ**: アジャイル開発
- **カスタマージャーニーマップ**: UXデザイン
- **マインドマップ**: 思考整理
- **SWOT分析**: 戦略策定
- **レトロスペクティブ**: ふりかえり
- **ビジネスモデルキャンバス**: ビジネス設計

### 5. 投票・タイマー
- **投票機能**: アイデアに投票
- **タイマー**: ワークショップのタイムボックス
- **プレゼンテーションモード**: ステークホルダーへの共有

### 6. 統合・拡張
- **Jira**: チケット連携
- **Confluence**: ドキュメント埋め込み
- **Slack**: 通知連携
- **Microsoft Teams**: チャット統合
- **Google Workspace**: Drive、Docs連携
- **Figma**: デザイン統合
- **Asana、Trello**: タスク管理連携

## 利用方法

### サインアップ

```bash
1. 公式サイトにアクセス
   https://miro.com/

2. Sign up for free
   - Google、Microsoft、Slack、メールアドレス

3. ワークスペース作成
   - チーム名設定
   - メンバー招待
```

### 基本的なボード作成

```
1. 新規ボード作成
   Dashboard → New board
   または
   Template → カテゴリ選択 → テンプレート選択

2. 付箋追加
   左サイドバー → Sticky notes
   カラー選択、テキスト入力

3. 図形追加
   左サイドバー → Shapes
   四角形、円、線、矢印等

4. コネクター追加
   オブジェクト選択 → 端点からドラッグ → 別オブジェクトへ接続

5. テキスト追加
   左サイドバー → Text → キャンバスにクリック
```

### ブレインストーミングワークショップ

```
1. テンプレート選択
   Template → Brainstorming → Brainwriting

2. タイマー設定
   上部メニュー → Timer → 5分設定 → Start

3. 参加者がアイデアを付箋で追加
   各自がカラー付箋を使用

4. グルーピング
   類似アイデアをまとめる

5. 投票
   上部メニュー → Voting → 各自3票で投票

6. 優先度決定
   投票結果を集計、上位アイデアを選定
```

### アジャイル開発（カンバンボード）

```
1. テンプレート選択
   Template → Agile → Kanban Board

2. 列設定
   Backlog、To Do、In Progress、In Review、Done

3. カード追加
   Backlog列に新規カード（付箋またはカード）追加
   - タイトル
   - 説明
   - 担当者（@メンション）

4. カードの移動
   タスク進捗に応じてドラッグ&ドロップ

5. Jira連携
   左サイドバー → Integrations → Jira
   Jiraチケットを同期
```

### ユーザーストーリーマッピング

```
1. テンプレート選択
   Template → Agile → User Story Map

2. エピック定義
   横軸にエピック（大機能）を配置

3. ユーザーストーリー追加
   各エピック下にストーリーを縦に配置

4. 優先度付け
   上から順に優先度高

5. リリース計画
   横線でリリース境界を区切る
```

### チーム招待・権限管理

```
1. メンバー招待
   右上 → Share → Invite team members
   メールアドレス入力

2. 権限設定
   - Can edit: 編集可能
   - Can comment: コメントのみ
   - Can view: 閲覧のみ

3. リンク共有
   Share → Copy link
   リンクから誰でもアクセス可能（設定次第）
```

### エクスポート

```
1. 画像エクスポート
   右上 → Export → Image (PNG, JPG)
   フレーム選択またはボード全体

2. PDFエクスポート
   Export → PDF
   全体または選択フレーム

3. CSVエクスポート（テーブルのみ）
   テーブル選択 → Export as CSV
```

## エディション・料金

| エディション | 価格（月額） | 特徴 |
|-------------|-------------|------|
| **Free** |  無料 | 3ボード、無制限メンバー、基本機能 |
| **Starter** |  $8/ユーザー | 無制限ボード、ゲストエディター、プライベートボード |
| **Business** |  $16/ユーザー | SSO、高度な権限、Jira統合、Admin管理 |
| **Enterprise** |  要問い合わせ | 専用サポート、カスタムSLA、監査ログ、データガバナンス |

### 無料プランの制約
- ボード数: 3個まで
- ゲストエディター: 不可
- プライベートボード: 不可
- 高度な統合: 制限あり

## メリット

###  主な利点

1. **直感的UI**: ドラッグ&ドロップで簡単操作
2. **リアルタイム協業**: 複数ユーザーの同時編集
3. **豊富なテンプレート**: 150+の業務テンプレート
4. **無限キャンバス**: 制限なしの作業スペース
5. **リモートワーク対応**: ビデオチャット統合
6. **統合機能**: Jira、Slack、Teams等との連携
7. **投票・タイマー**: ワークショップ運営機能
8. **モバイルアプリ**: iOS、Android対応
9. **アクセシビリティ**: ブラウザからすぐ利用可能
10. **アクティブ開発**: 継続的な機能追加

## デメリット

###  制約・課題

1. **無料プラン制限**: 3ボードまで
2. **オフライン不可**: インターネット接続必須
3. **パフォーマンス**: 大規模ボードで動作遅延
4. **データロックイン**: Miro形式、他ツールへの移行困難
5. **コスト**: チーム全体で使うと高額
6. **学習コスト**: 高度な機能は習得に時間
7. **セキュリティ**: エンタープライズ版以外はSOC2限定的
8. **カスタマイズ**: テンプレートのカスタマイズ性に限界

## 代替ツール

| **Mural** | ビジュアル協業、ファシリテーション | Miroと類似、ワークショップ特化 |
