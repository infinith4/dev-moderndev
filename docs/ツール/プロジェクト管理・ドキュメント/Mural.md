# Mural

## 概要

Muralは、ビジュアルコラボレーションとファシリテーションに特化したオンラインホワイトボードプラットフォームです。無限のキャンバスでチームがブレインストーミング、デザイン思考、アジャイルプランニング、ワークショップを実施できます。豊富なテンプレート、タイマー、投票機能、ファシリテーター機能により、リモート・ハイブリッドチームの生産的なコラボレーションを支援します。

## 主な機能

### 1. 無限キャンバス
- **無制限ワークスペース**: 無限に広がるホワイトボード
- **ズーム・パン**: スムーズなナビゲーション
- **エリア**: 領域を区切って整理
- **アウトライン**: 全体構造の把握

### 2. リアルタイム協業
- **同時編集**: 複数ユーザーが同時に作業
- **カーソル表示**: 他メンバーの操作を可視化
- **コメント**: オブジェクトへのコメント
- **ビデオチャット**: Zoom、Teams統合

### 3. 豊富なオブジェクト
- **付箋**: ブレインストーミング
- **図形**: アイコン、矢印、フレーム
- **画像・ファイル**: ドラッグ&ドロップ
- **テキスト**: 見出し、本文
- **チャート**: グラフ、表

### 4. ファシリテーター機能
- **プライベートモード**: 参加者の編集を一時制限
- **サモン**: 全員を特定エリアに移動
- **タイマー**: ワークショップのタイムボックス
- **投票**: アイデアに投票
- **アイスブレイク**: チームビルディング

### 5. テンプレート
- **300+テンプレート**: 業務用テンプレート
- **ブレインストーミング**: アイデア整理
- **アジャイル**: カンバン、レトロスペクティブ
- **デザイン思考**: カスタマージャーニーマップ
- **戦略策定**: SWOT、ビジネスモデルキャンバス

### 6. 統合・エクスポート
- **Microsoft Teams**: 統合
- **Slack**: 通知連携
- **Jira**: チケット連携
- **PNG/PDF**: エクスポート

## 利用方法

### サインアップ

```
1. https://www.mural.co/ にアクセス
2. Sign up for free
3. メールアドレス、Google、Microsoft認証
4. ワークスペース作成
```

### 基本操作

```
1. 新規Mural作成
   + Create mural → Blank canvas

2. 付箋追加
   左ツールバー → Sticky notes → キャンバスにクリック

3. テキスト追加
   左ツールバー → Text → キャンバスにドラッグ

4. 画像追加
   ドラッグ&ドロップ または Upload

5. アイコン追加
   左ツールバー → Icons → 検索してキャンバスに配置
```

### ワークショップ運営

```
# ブレインストーミングセッション
1. テンプレート選択: Brainstorming

2. タイマー設定
   上部メニュー → Timer → 5分 → Start

3. 参加者がアイデアを付箋で追加

4. グルーピング
   類似アイデアをまとめる

5. 投票
   上部メニュー → Vote → 各自3票

6. 優先度決定
   投票結果を集計
```

### ファシリテーター機能

```
# プライベートモード
1. 上部メニュー → Private mode ON
2. 参加者の編集を一時制限
3. ファシリテーターが説明

# サモン機能
1. 特定エリアにズーム
2. 上部メニュー → Summon everyone
3. 全員が同じエリアに移動
```

### アジャイル開発（レトロスペクティブ）

```
1. テンプレート: Agile Retrospective

2. タイムライン
   - What went well?
   - What could be improved?
   - Action items

3. 参加者がそれぞれのエリアに付箋追加

4. グルーピング・投票

5. アクションアイテム決定
```

## メリット

### ✅ 主な利点

1. **ファシリテーター機能**: プライベートモード、サモン
2. **無限キャンバス**: 制限なしの作業スペース
3. **豊富なテンプレート**: 300+の業務テンプレート
4. **投票・タイマー**: ワークショップ運営機能
5. **リアルタイム協業**: 同時編集
6. **直感的UI**: 習得容易
7. **統合**: Teams、Slack連携
8. **モバイルアプリ**: iOS、Android対応
9. **エクスポート**: PNG、PDF出力
10. **コミュニティ**: 豊富なリソース

## デメリット

### ❌ 制約・課題

1. **無料プラン制限**: 3 Muralsまで
2. **コスト**: Miroより高価
3. **オフライン不可**: インターネット接続必須
4. **パフォーマンス**: 大規模Muralで遅延
5. **データロックイン**: Mural固有形式
6. **カスタマイズ**: テンプレートのカスタマイズに限界
7. **学習曲線**: 高度な機能は習得に時間
8. **統合**: MiroよりAPI制限的

## 代替ツール

| ツール | 特徴 | 比較 |
|--------|------|------|
| **Miro** | ビジュアル協業、豊富な統合 | Muralと類似、Miroの方が統合多い |
| **FigJam** | Figma製ホワイトボード | Muralよりデザイナー向け |
| **Microsoft Whiteboard** | Microsoft 365統合 | MuralよりシンプルだがTeams統合強い |
| **Lucidspark** | アイデア整理、投票 | Muralと類似 |
| **Notion** | ドキュメント+ホワイトボード | Muralよりドキュメント重視 |
