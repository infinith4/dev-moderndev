# Notion

## 概要

Notion は、サンフランシスコに本社を置く Notion Labs が提供するオールインワンのワークスペース・コラボレーションツールです。ノート、タスク管理、Wiki、データベース、カレンダーを統合し、「コネクテッドワークスペース」として個人からチームまで、情報管理とプロジェクト管理を一元化します。ブロックベースの柔軟なエディタ、リレーショナルデータベース、豊富なテンプレートにより、世界中で3,000万人以上のユーザーに利用されています。

## 基本的な使い方

### 1. ワークスペースの作成

```
1. Notion にサインアップ
2. ワークスペース名を入力
3. 用途を選択
   - 個人利用
   - チーム
   - 会社
4. テンプレートを選択（またはスキップ）
5. ワークスペースが作成される
```

### 2. ページの作成

```
# 方法1: サイドバーから
1. 左サイドバーの「+」をクリック
2. ページタイトルを入力
3. アイコン・カバー画像を追加（オプション）

# 方法2: ページ内に作成
1. 既存のページを開く
2. /page と入力
3. Enter でサブページ作成

# ページの移動
- ドラッグ&ドロップで階層を変更
- サイドバーで整理
```

### 3. ブロックの使用

```
# テキストブロック
/heading1, /h1       → 見出し1
/heading2, /h2       → 見出し2
/text                → テキスト
/bullet              → 箇条書き
/numbered            → 番号付きリスト
/todo                → チェックボックス
/toggle              → トグルリスト
/quote               → 引用
/code                → コードブロック

# メディアブロック
/image               → 画像
/video               → 動画
/file                → ファイル
/pdf                 → PDF
/audio               → オーディオ

# 埋め込みブロック
/embed               → 一般的な埋め込み
/figma               → Figma
/github              → GitHub Gist
/google              → Google ドライブ
/maps                → Google Maps
/miro                → Miro
/twitter             → Twitter

# データベース
/table               → テーブル
/board               → ボード
/list                → リスト
/calendar            → カレンダー
/gallery             → ギャラリー
/timeline            → タイムライン

# 高度なブロック
/link                → ページリンク
/mention             → メンション
/equation            → 数式
/divider             → 区切り線
/toc                 → 目次
/breadcrumb          → パンくずリスト
```

### 4. データベースの作成

```
1. ページを開く
2. /table（またはボード、リスト等）と入力
3. データベース名を入力
4. プロパティ（カラム）を追加
   - 右端の「+」をクリック
   - プロパティタイプを選択
   - プロパティ名を入力
5. エントリー（行）を追加
   - 「+ 新規」をクリック
   - データを入力
```

### 5. ビューの追加

```
1. データベースを開く
2. 左上の「+ ビューを追加」
3. ビュータイプを選択
   - テーブル
   - ボード
   - リスト
   - カレンダー
   - ギャラリー
   - タイムライン
4. ビュー名を入力
5. フィルター・ソート・グループ化を設定

# 例: タスクデータベース
ビュー1: すべてのタスク（テーブル）
ビュー2: ステータス別（ボード）
ビュー3: 期限順（カレンダー）
ビュー4: 担当者別（グループ化リスト）
```

```
# リレーションの作成
1. プロパティを追加 → 「リレーション」
2. リンク先のデータベースを選択
3. エントリー間でリンクを作成

# ロールアップ（集計）
1. プロパティを追加 → 「ロールアップ」
2. リレーションプロパティを選択
3. 集計するプロパティを選択
4. 計算方法を選択
   - カウント
   - 合計
   - 平均
   - 最小/最大
   - etc.

# 例: プロジェクトとタスク
プロジェクトDB
├── タスク（リレーション → タスクDB）
├── 完了タスク数（ロールアップ: タスク.ステータス = 完了 のカウント）
└── 総工数（ロールアップ: タスク.工数 の合計）

タスクDB
├── プロジェクト（リレーション → プロジェクトDB）
└── プロジェクト期限（ロールアップ: プロジェクト.期限）
```

### フォーミュラ（数式）

```
# 基本的な計算
prop("予算") - prop("実績")

# 条件分岐
if(prop("ステータス") == "完了", "✅", "⏳")

# 日付計算
dateBetween(prop("期限"), now(), "days")

# 文字列操作
concat(prop("姓"), " ", prop("名"))

# 複雑な例: 進捗率
floor(prop("完了タスク数") / prop("総タスク数") * 100) + "%"

# ステータスに応じた色分け
if(prop("進捗") >= 100, "🟢",
  if(prop("進捗") >= 50, "🟡", "🔴"))
```

## API 利用

### REST API

```javascript
// Node.js での使用
const { Client } = require('@notionhq/client');

const notion = new Client({ auth: process.env.NOTION_API_KEY });

// データベースのクエリ
const response = await notion.databases.query({
  database_id: 'DATABASE_ID',
  filter: {
    property: 'Status',
    select: {
      equals: '進行中'
    }
  },
  sorts: [
    {
      property: '期限',
      direction: 'ascending'
    }
  ]
});

// ページの作成
await notion.pages.create({
  parent: {
    database_id: 'DATABASE_ID'
  },
  properties: {
    Name: {
      title: [
        {
          text: {
            content: '新しいタスク'
          }
        }
      ]
    },
    Status: {
      select: {
        name: '未着手'
      }
    },
    Assignee: {
      people: [
        {
          id: 'USER_ID'
        }
      ]
    }
  }
});

// ページの更新
await notion.pages.update({
  page_id: 'PAGE_ID',
  properties: {
    Status: {
      select: {
        name: '完了'
      }
    }
  }
});
```

### Python での使用

```python
from notion_client import Client

notion = Client(auth="NOTION_API_KEY")

# データベースのクエリ
results = notion.databases.query(
    database_id="DATABASE_ID",
    filter={
        "property": "Status",
        "select": {
            "equals": "進行中"
        }
    }
)

for page in results["results"]:
    print(page["properties"]["Name"]["title"][0]["text"]["content"])
```
