# xUnit

## 概要

xUnitは、ソフトウェアテスティングフレームワークの総称（xUnit系アーキテクチャ）です。Kent Beckが考案したSUnitに起源を持ち、JUnit（Java）、NUnit（.NET）、pytest（Python）、xUnit.net（.NET）などの各言語向けフレームワークの基盤となる設計パターンです。

.NET向けの具体的な実装については [xUnit.net](./xUnit_net.md) を参照してください。
