# LibreOffice

## 概要
LibreOffice は文書・表計算・プレゼンテーションを扱えるオープンソースのオフィススイートです。帳票作成やデータ集計を低コストで運用したい場合に有効です。

## 料金
| プラン | 料金 | 備考 |
|---|---:|---|
| LibreOffice | 無料 | OSS、追加ライセンス不要 |
| 商用サポート | 有料 | ベンダー契約時のみ |

## 主な特徴
| 特徴 | 内容 |
|---|---|
| 無償利用 | ライセンスコストを抑えやすい |
| 互換性 | 一般的な文書・表計算形式を扱える |
| マルチプラットフォーム | Windows/macOS/Linux で利用可能 |

## 主な機能

### コア機能

| 機能 | 用途 | 実務での使いどころ |
|---|---|---|
| 表計算 | 集計・分析 | 日次実績、運用レポート |
| 文書作成 | 帳票テンプレート作成 | 報告書、手順書 |
| PDF 出力 | 配布用形式への変換 | 監査提出、共有資料 |

### 連携機能

| 機能 | 説明 |
|------|------|
| ツール連携 | 既存の開発・運用ツールと連携しやすい |
| 自動実行 | ローカル実行と CI 実行の両方に対応しやすい |

### 運用機能

| 機能 | 説明 |
|------|------|
| 結果管理 | 実行結果を蓄積して比較しやすい |
| 変更追跡 | 設定変更の影響を確認しやすい |

## インストールとセットアップ

公式URL:
- [公式リンク1](https://www.libreoffice.org/)
- [公式リンク2](https://documentation.libreoffice.org/)

セットアップ手順:
1. 利用 OS に合わせて導入します。
2. テンプレート保存ルールと共有先を決めます。
3. 互換性確認が必要なファイル形式を事前に検証します。

## 基本的な使い方
1. 帳票テンプレートを作成して共通フォーマット化します。
2. 入力項目と計算式を明確にし、誤入力を防止します。
3. 完成帳票を PDF 化して配布・保管します。

最小実行例:
- ローカル環境で LibreOffice の基本機能を実行
- CI で定期実行し、結果を比較

## メリット

|---|---|
| コスト | 無償で導入しやすい |
| 柔軟性 | 小規模運用に合わせて使いやすい |

## デメリット
| 観点 | 内容 |
|---|---|
| 互換差異 | 複雑書式で互換性課題が出る場合がある |
| 標準化 | 手作業運用では品質がばらつきやすい |
| 自動化 | 大規模帳票運用では専用基盤が必要になる |

## 他ツールとの比較
| ツール | 強み | 使い分け |
|---|---|---|
| LibreOffice | 無償導入と基本機能が充実 | 小中規模の帳票運用 |
| Microsoft Excel | 企業標準導入と周辺連携が豊富 | 既存 Microsoft 環境 |
| JasperReports | 自動帳票生成に強い | システム連携型運用 |

## ベストプラクティス
1. テンプレートと入力ルールを分離し、更新統制を行います。
2. 出力形式は PDF を基準にし、配布品質を安定させます。
3. 手作業工程はチェックリスト化して属人化を抑えます。

## まとめ

1. **導入容易** : LibreOffice は低コストで帳票・データ処理を開始しやすい実用的な選択肢です。
2. **標準化** : 継続運用にはテンプレート管理と手順標準化が重要です。
3. **拡張性** : 大量処理が必要になった段階で自動化ツールとの併用を検討するのが現実的です。
