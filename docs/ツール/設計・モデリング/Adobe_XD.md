# Adobe XD（画面設計）

## 概要

Adobe XDは、Adobe製のUI/UXデザインツールで、画面設計、プロトタイプ作成、デザインシステム構築に特化しています。Adobe Creative Cloudとの統合により、Photoshop、Illustratorとシームレスに連携でき、基本設計フェーズでの詳細な画面仕様書作成に適しています。

### 基本設計フェーズでの活用

- **画面レイアウト**: ピクセルパーフェクトな画面設計
- **インタラクティブプロトタイプ**: 画面遷移とアニメーションの可視化
- **デザインスペック**: 開発者向けの詳細仕様自動生成
- **コンポーネント**: 再利用可能なUIパーツの管理
- **レスポンシブリサイズ**: マルチデバイス対応の画面設計

### 料金プラン

- **無料スタータープラン**: 基本機能（共有ドキュメント1つまで）
- **単体プラン**: ¥1,298/月（XDのみ）
- **Creative Cloudコンプリートプラン**: ¥6,480/月（全Adobeアプリ）

### メリット・デメリット

**メリット**
- Adobe Creative Cloudとの統合（Photoshop、Illustrator等）
- 高品質なデザインツール
- デザインスペック自動生成で開発者への引き渡しが容易
- リピートグリッド機能で効率的なデザイン
- 音声プロトタイプ対応

**デメリット**
- 有料（無料版は機能制限あり）
- Figmaに比べてリアルタイム協業機能が劣る
- 開発が停滞気味（2023年以降、大きなアップデートなし）

## 利用方法

### 1. Adobe XDの起動と新規プロジェクト作成

1. Adobe XDを起動
2. ホーム画面で「新規作成」をクリック
3. プリセットサイズを選択:
   - Web: 1920 x 1080、1366 x 768
   - iPhone: 375 x 812（iPhone 13）
   - iPad: 834 x 1194
   - Android: 360 x 640
   - カスタム: 任意のサイズ

### 2. アートボード（画面）の作成

#### アートボードの追加

1. 左ツールバーから「アートボード」ツール（A）を選択
2. キャンバス上でドラッグしてサイズを指定
3. または、右パネルからプリセットサイズを選択
4. アートボード名を入力（例: "01_ログイン画面"）

#### 例: 会員登録フォームの画面設計

**アートボード名**: 02_会員登録画面
**サイズ**: Web 1366 x 768

### 3. UI要素の配置

#### 基本図形ツール

- **Rectangle** (R): 四角形（ボタン、入力フィールド等）
- **Ellipse** (E): 円形（アイコン、プロフィール画像等）
- **Polygon**: 多角形
- **Line** (L): 直線
- **Pen** (P): ベジェ曲線

#### テキストツール

1. テキストツール（T）を選択
2. クリックまたはドラッグでテキストエリアを作成
3. テキストを入力
4. 右パネルでフォント、サイズ、色、行間を設定

### 4. 画面レイアウトの具体例

#### 例: 会員登録画面

**1. ヘッダーの作成**

1. Rectangle (R) を作成: 1366 x 80
2. 塗りつぶし: #2C3E50
3. テキスト追加: "サイト名"（24px、Bold、White）

**2. 登録フォームの作成**

中央にフォームコンテナを配置:

| 要素 | サイズ | プロパティ |
|------|--------|----------|
| フォームコンテナ | 500 x 700 | 背景: White、角丸: 8px、Shadow |
| タイトル | - | "会員登録"（32px、Bold） |
| 氏名ラベル | - | "氏名"（14px） |
| 氏名入力 | 460 x 48 | Border: 1px #CCC、角丸: 4px |
| メールラベル | - | "メールアドレス"（14px） |
| メール入力 | 460 x 48 | Border: 1px #CCC、角丸: 4px |
| パスワードラベル | - | "パスワード"（14px） |
| パスワード入力 | 460 x 48 | Border: 1px #CCC、角丸: 4px |
| 登録ボタン | 460 x 48 | 背景: #3498DB、文字: White（16px、Bold） |

**3. 入力フィールドのステータス表示**

エラー状態の表現:
- Border色を#E74C3Cに変更
- エラーメッセージを赤字で表示（12px、#E74C3C）

### 5. リピートグリッド

リピートグリッド機能により、リスト表示を効率的に作成できます。

#### 例: 商品一覧

1. 商品カード（1つ）を作成:
   - 商品画像: 200 x 200
   - 商品名: "商品名"（16px）
   - 価格: "¥10,000"（18px、Bold）
   - カートボタン: 200 x 40

2. 商品カードを選択→右クリック→「リピートグリッド」
3. ハンドルをドラッグして、グリッドを拡張（例: 3行 x 4列）
4. 間隔を調整（水平: 20px、垂直: 20px）

リピートグリッド内のテキストや画像は、CSVデータやフォルダ内の画像で一括置換可能です。

### 6. コンポーネントの作成

再利用可能なUIパーツをコンポーネント化します。

#### 例: ボタンコンポーネント

1. ボタン（Rectangle + Text）を作成
2. グループ化（Cmd/Ctrl + G）
3. 右クリック→「コンポーネントにする」（Cmd/Ctrl + K）
4. コンポーネント名: "Primary Button"

#### ステート（状態）の追加

1. コンポーネントを選択
2. 右パネル「コンポーネント」→「ステートを追加」
3. ステート名を入力:
   - Default（デフォルト）
   - Hover（ホバー時）
   - Active（クリック時）
   - Disabled（無効時）

4. 各ステートでスタイルを変更:
   - Default: 背景 #3498DB
   - Hover: 背景 #2980B9
   - Active: 背景 #1F618D
   - Disabled: 背景 #95A5A6

### 7. プロトタイプモード

画面遷移とインタラクションを定義します。

#### 画面遷移の設定

1. 上部タブで「プロトタイプ」を選択
2. 遷移元のオブジェクト（例: 登録ボタン）をクリック
3. 青い矢印ハンドルを遷移先アートボードまでドラッグ
4. インタラクション設定:
   - **トリガー**: タップ、ドラッグ、ホバー、時間
   - **アクション**: 遷移、オーバーレイ、自動アニメーション
   - **イージング**: イーズアウト、イーズイン
   - **デュレーション**: 0.3秒

#### 自動アニメーション

画面間の要素が自動的にアニメーションします。

1. 遷移元と遷移先で同じレイヤー名を使用
2. アクションを「自動アニメーション」に設定
3. プレビューで確認

### 8. 画面項目定義の記載

Adobe XDでは、項目定義をテキストレイヤーやコメントで記載します。

#### 方法1: レイヤー名で定義

各入力フィールドのレイヤー名に属性を記載:

```
email_input [text, 100桁, 必須, メール形式]
password_input [password, 20桁, 必須, 8文字以上]
name_input [text, 50桁, 必須]
```

#### 方法2: アートボード横に定義表を作成

画面の横にテキストボックスで項目定義表を作成:

```
【画面項目定義】
■ 氏名 (name)
- 型: text
- 桁数: 50
- 必須: ○
- DB項目: user_name

■ メールアドレス (email)
- 型: email
- 桁数: 100
- 必須: ○
- バリデーション: RFC 5322準拠
- DB項目: email

■ パスワード (password)
- 型: password
- 桁数: 8-20
- 必須: ○
- バリデーション: 英数字混在必須
- DB項目: password_hash
```

### 9. デザインスペックの共有

開発者に正確な仕様を伝えるために、デザインスペックを生成します。

#### 共有リンクの作成

1. 右上の「共有」ボタンをクリック
2. 「開発用に共有」を選択
3. 「リンクを作成」をクリック
4. リンクをコピーして開発者に送信

#### デザインスペックで確認できる情報

開発者がリンクを開くと、以下の情報を確認できます:

- **距離**: 要素間の距離（px単位）
- **サイズ**: 幅・高さ
- **カラー**: 16進数カラーコード
- **フォント**: フォント名、サイズ、行間、文字間隔
- **CSS**: CSSコードスニペット
- **アセット**: 画像やアイコンのダウンロード

### 10. プラグインの活用

Adobe XDには豊富なプラグインがあります。

#### おすすめプラグイン

**1. Stark（アクセシビリティチェック）**

1. Plugins → Discover Plugins → 「Stark」をインストール
2. 色のコントラストをチェック
3. WCAG 2.1準拠を確認

**2. Lorem Ipsum（ダミーテキスト生成）**

1. テキストレイヤーを選択
2. Plugins → Lorem Ipsum → Generate
3. ダミーテキストが挿入される

**3. UI Faces（プロフィール画像）**

1. 円形のプレースホルダーを作成
2. Plugins → UI Faces → Generate
3. ランダムなプロフィール画像が挿入

**4. Icons 4 Design（アイコン挿入）**

1. Plugins → Icons 4 Design
2. アイコンを検索（例: "cart", "user"）
3. アイコンを挿入

### 11. レスポンシブリサイズ

マルチデバイス対応の画面設計にはレスポンシブリサイズを活用します。

#### リサイズの設定

1. オブジェクトを選択
2. 右パネル「レスポンシブリサイズ」で設定:
   - **固定幅**: 幅を固定
   - **固定高さ**: 高さを固定
   - **左右に拡大**: 親要素に合わせて水平方向に拡大
   - **上下に拡大**: 親要素に合わせて垂直方向に拡大

#### 例: ヘッダーのレスポンシブ設定

- ヘッダー全体: 左右に拡大（画面幅に合わせる）
- ロゴ: 固定幅・固定高さ
- ナビゲーションメニュー: 左右に拡大

### 12. カラーとフォントのアセット管理

#### カラーアセット

1. 塗りつぶし色を選択
2. カラーピッカーの「+」アイコンをクリック
3. カラーがアセットに追加される
4. 名前を付ける（例: "Primary Color", "Error Color"）

#### 文字スタイル

1. テキストを選択
2. 右パネル「文字」→「+」アイコン
3. 文字スタイル名を入力（例: "H1 - Heading", "Body Text"）
