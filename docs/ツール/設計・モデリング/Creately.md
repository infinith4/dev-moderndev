# Creately

## 概要

Createlyは、クラウドベースのビジュアルコラボレーションプラットフォームです。フローチャート、ER図、UML、ワイヤーフレーム、ネットワーク図等、70+のダイアグラムタイプをサポートし、リアルタイムチーム協業を実現します。AI搭載のスマート図形配置、無限キャンバス、データリンク機能により、複雑なビジネスプロセスやシステムアーキテクチャを可視化します。

## 主な機能

### 1. 多様なダイアグラム
- **フローチャート**: プロセスフロー
- **ER図**: データベース設計
- **UML**: クラス図、シーケンス図、ユースケース図
- **ワイヤーフレーム**: UI設計
- **ネットワーク図**: インフラ構成
- **組織図**: 組織構造
- **マインドマップ**: ブレインストーミング

### 2. リアルタイム協業
- **同時編集**: 複数ユーザー同時作業
- **カーソル表示**: 他メンバーの操作可視化
- **コメント**: 図形へのコメント
- **ビデオチャット**: 統合ビデオ通話
- **プレゼンスインジケーター**: オンライン状態表示

### 3. スマート機能
- **AI配置**: 自動整列・配置
- **スマートコネクター**: 自動接続
- **テンプレート**: 1,000+テンプレート
- **図形ライブラリ**: AWS、Azure、GCP等

### 4. データリンク
- **データインポート**: Excel、CSV
- **双方向同期**: データ変更を図に反映
- **データベース**: 外部データソース連携

### 5. エクスポート・統合
- **エクスポート**: PNG、PDF、SVG
- **Confluence**: マクロで埋め込み
- **Jira**: チケットリンク
- **Slack**: 通知連携
- **Google Drive**: ストレージ統合

## 利用方法

### 基本操作

```
1. 新規ダイアグラム作成
   Create → Select template または Blank canvas

2. 図形追加
   左パネル → Shapes
   - Drag & Drop で配置

3. コネクター追加
   図形の端点からドラッグ → 別図形へ接続

4. テキスト追加
   図形ダブルクリック → テキスト入力
```

### フローチャート作成

```
1. テンプレート: Flowchart → Blank Flowchart
2. 図形追加:
   - Start/End (楕円)
   - Process (長方形)
   - Decision (ひし形)
3. コネクター: 矢印で接続
4. ラベル: Yes/No ラベル追加
```

### ER図作成

```
1. テンプレート: ER Diagram
2. エンティティ追加:
   - Entity: User
   - Attributes: id, name, email
3. リレーションシップ:
   - One to Many: User → Posts
   - Many to Many: Posts ← → Tags
4. カーディナリティ: 1:N、M:N設定
```

### AWSアーキテクチャ図

```
1. テンプレート: AWS Architecture
2. 図形ライブラリ: AWS Icons
3. コンポーネント配置:
   - VPC
   - EC2
   - RDS
   - S3
   - CloudFront
4. ネットワーク接続: セキュリティグループ、サブネット
```

### チームコラボレーション

```
1. Share → Invite team members
2. Email入力 → Send invitation
3. 権限設定:
   - Can edit
   - Can comment
   - Can view
4. リアルタイム共同編集開始
```

### Confluence統合

```
1. Confluenceページ編集
2. "/" → Creately
3. マクロ挿入
4. ダイアグラム選択または新規作成
5. ページ保存 → 自動レンダリング
```

## メリット

### ✅ 主な利点

1. **リアルタイム協業**: 同時編集
2. **AI配置**: 自動整列・配置
3. **豊富なテンプレート**: 1,000+テンプレート
4. **クラウドベース**: ブラウザで利用
5. **図形ライブラリ**: AWS、Azure、GCP等
6. **データリンク**: 外部データ連携
7. **Confluence統合**: シームレス統合
8. **無料プラン**: 基本機能利用可能
9. **モバイルアプリ**: iOS、Android
10. **直感的UI**: 学習容易

## デメリット

### ❌ 制約・課題

1. **無料プラン制限**: 3ドキュメントのみ
2. **オフライン不可**: インターネット必須
3. **パフォーマンス**: 大規模図で遅延
4. **カスタマイズ**: 図形カスタマイズに限界
5. **バージョン管理**: Git統合なし
6. **エクスポート制限**: 一部形式は有料
7. **コスト**: Lucidchartより高い場合あり
8. **学習曲線**: 高度な機能は習得に時間

## 代替ツール

| ツール | 特徴 | 比較 |
|--------|------|------|
| **Lucidchart** | クラウド作図、豊富な統合 | Createlyと類似 |
| **draw.io (diagrams.net)** | 無料、オープンソース | Createlyより無料だが機能少ない |
| **Microsoft Visio** | デスクトップ、エンタープライズ | Createlyよりデスクトップ重視 |
| **Miro** | ホワイトボード、協業 | Createlyよりブレインストーミング向け |
| **PlantUML** | テキストベース | Createlyよりコード重視 |
