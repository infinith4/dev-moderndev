# Lucidchart

## 概要

Lucidchartは、クラウドベースの図表作成ツールで、ビジネスプロセス図、フローチャート、組織図、BPMNなどの作成に対応しています。ブラウザベースでリアルタイムの協業が可能な点が特徴です。

### 主な特徴

- **ブラウザベース**: インストール不要で、ブラウザからアクセス可能
- **リアルタイム協業**: 複数ユーザーが同時に編集可能
- **テンプレート豊富**: 様々な業務に対応したテンプレートを提供
- **BPMN対応**: BPMN 2.0に対応したプロセス図の作成が可能
- **統合機能**: Google Workspace、Microsoft 365、Slack等との連携

### 料金プラン

- **Free版**: 基本機能（制限あり）
- **Individual**: $7.95/月（個人向け）
- **Team**: $9/月/ユーザー（チーム向け）
- **Enterprise**: カスタム価格（大企業向け）

### メリット・デメリット

**メリット**
- ブラウザベースでどこからでもアクセス可能
- リアルタイム協業により、チームでの作業が効率的
- テンプレートが豊富で、すぐに作図を開始できる
- BPMN対応で標準的なプロセス図の作成が可能
- 各種クラウドサービスとの統合機能が充実

**デメリット**
- 無料版は機能制限あり（ドキュメント数、図形数など）
- 有料版がやや高額
- オフラインでは利用不可
- 複雑な図を作成すると動作が重くなることがある

## 利用方法

### 1. アカウント作成

1. [Lucidchart公式サイト](https://www.lucidchart.com/)にアクセス
2. 「Sign up free」をクリック
3. メールアドレスまたはGoogle/Microsoft アカウントで登録
4. アカウントを有効化

### 2. 新規ドキュメント作成

1. ダッシュボードから「+ New」ボタンをクリック
2. テンプレートを選択するか、「Blank Document」を選択
3. ドキュメント名を入力

### 3. BPMNテンプレートの利用

1. 「+ New」→「Templates」を選択
2. 検索バーに「BPMN」と入力
3. 適切なBPMNテンプレートを選択（例: "BPMN Process Map"）
4. 「Use Template」をクリック

### 4. 図形ライブラリから要素を配置

#### 図形ライブラリの表示

1. 左側のパネルで「Shapes」をクリック
2. 「BPMN 2.0」ライブラリを選択
3. 必要に応じて他のライブラリも追加（フローチャート、組織図など）

#### 要素の配置

1. **開始イベント**: 左パネルから「Start Event」（緑の円）をドラッグ&ドロップ
2. **タスク**: 「Task」（四角形）を配置
3. **ゲートウェイ**: 「Gateway」（菱形）を配置
4. **終了イベント**: 「End Event」（赤の円）を配置

### 5. コネクタで接続

1. 図形をクリックすると、四方に赤い点（コネクションポイント）が表示される
2. 赤い点をクリックして、接続先の図形までドラッグ
3. 自動的にシーケンスフロー（矢印）が作成される

### 6. プロセスフロー作成の具体例

**例: 購買承認プロセス**

```
[開始] → [購買申請] → [上司承認] → <承認？>
                                    ├─ Yes → [発注処理] → [納品確認] → [支払処理] → [終了]
                                    └─ No → [申請者へ差戻し] → [購買申請] (ループ)
```

#### 作成手順

1. キャンバスに開始イベントを配置
2. タスク「購買申請」を配置し、開始イベントから接続
3. タスク「上司承認」を配置
4. 排他ゲートウェイ「承認？」を配置
5. Yes経路の作成:
   - タスク「発注処理」→「納品確認」→「支払処理」→終了イベント
6. No経路の作成:
   - タスク「申請者へ差戻し」→「購買申請」へループバック
7. 各コネクタに条件ラベルを追加（Yes/No）

### 7. スタイル設定とカスタマイズ

1. 図形をクリックして選択
2. 右側のプロパティパネルでスタイルを設定
   - 塗りつぶし色
   - 線の色・太さ
   - フォント・文字サイズ
3. 複数の図形を選択して一括スタイル変更も可能

### 8. チーム共有と協業

1. 右上の「Share」ボタンをクリック
2. 共有方法を選択:
   - **メールで招待**: メールアドレスを入力して招待
   - **リンク共有**: 共有リンクを生成（閲覧のみ/編集可能）
3. 権限を設定（View/Edit）
4. 「Send」をクリック

### 9. エクスポート

1. 「File」→「Download As」を選択
2. 形式を選択:
   - PDF
   - PNG
   - JPEG
   - SVG
   - Visio (VDX)
3. オプションを設定（ページ範囲、品質など）
4. 「Download」をクリック
