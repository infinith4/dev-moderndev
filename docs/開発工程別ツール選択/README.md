## 📊 使い方

1. **工程選択**: 実施する開発工程のファイルを開く
2. **ツール選択**: 使用するツールにチェック ☑ を入れる
3. **詳細確認**: ツール詳細リンクから詳しい情報を確認
4. **プロジェクト記録**: 選択内容を `PROJECT_V1/プロジェクトツール選択.md` に記録
